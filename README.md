# hosts-switch (hs)

Простой скрипт для быстрого переключения локальных хостов при веб-разработке.

## Зачем это нужно?

Когда вы разрабатываете несколько сайтов локально, удобно добавлять их домены в `/etc/hosts` для доступа по HTTPS. Этот скрипт автоматически:

- Сканирует папку с сайтами (например, `/var/www/`)
- Показывает интерактивный список хостов с возможностью выбора
- Добавляет выбранные домены в `/etc/hosts` (на `127.0.0.1`)
- При повторном запуске — показывает активные домены и удаляет выбранные

**Один запуск — включено, второй — выключено.**

---

## Требования

- Linux (проверено на Manjaro/Arch)
- Bash
- Права root (sudo)
- [fzf](https://github.com/junegunn/fzf) — для интерактивного выбора
```bash
# Установка fzf
sudo apt install fzf       # Debian/Ubuntu
sudo pacman -S fzf         # Arch/Manjaro
brew install fzf           # macOS
```

## Установка
```bash
# Скачать скрипт
sudo curl -L https://github.com/santyago52/hosts-switch/raw/main/hs -o /usr/local/bin/hs

# Сделать исполняемым
sudo chmod +x /usr/local/bin/hs
```

Или скопируйте файл `hs` из этого репозитория в `/usr/local/bin/`.

---

## Использование
```bash
sudo hs
```

При запуске откроется интерактивный список хостов:

- **Пробел** — снять / поставить отметку
- **Enter** — применить
- **Esc** — выйти без изменений

**Первый запуск** — все найденные хосты помечены для добавления в `/etc/hosts`. Снимите отметку с тех, которые не нужны, и нажмите Enter.

**Повторный запуск** — показываются только активные домены, все помечены для удаления. Снимите отметку с тех, которые хотите оставить, и нажмите Enter.

---

## Настройка

Откройте скрипт и измените параметры в начале файла:
```bash
# Путь к папке с сайтами
WEB_ROOT="/var/www/"

# Папки-исключения (не будут добавлены в hosts)
EXCLUDE=("logs" "backup" ".git" "cache" "tmp" "html")
```

---

## Как это работает?

1. Скрипт сканирует `/var/www/` и получает список папок
2. Исключает папки из списка `EXCLUDE`
3. Пропускает папки с пробелами в имени (невалидные домены)
4. Открывает интерактивный список через `fzf` — все хосты отмечены по умолчанию
5. Для каждого выбранного домена добавляет в `/etc/hosts`:
   - `домен` → `127.0.0.1`
   - `www.домен` → `127.0.0.1`
6. При повторном запуске — удаляет только выбранные записи, остальные остаются

---

## Пример /etc/hosts после запуска
```
# BEGIN hs managed entries
127.0.0.1 site.com www.site.com site2.com www.site2.com
# END hs managed entries
```
